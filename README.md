## Tech Stack

**Client:** HTML, CSS, JS, TailwindCSS

**Local Server:** Node


## Run Locally

Clone the project

```bash
  git clone https://github.com/Eth3ri0n/Biarritz.git
```

Go to the project directory

```bash
  cd Biarritz
```

Install dependencies

```bash
  npm install
```

Start Tailwindcss

```bash
  npm run start
```
